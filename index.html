    VGA Controller

    Project

    In this Project a simple text system is developed on a FPGA device. The default resolution will be 640x480 but it can be reconfigured to support different resolutions, too.

    n this Project we will develop a pong game and display it on a monitor. We will carry the project on the FPGA and will send the video to the monitor via VGA interface. Our video resolution will be 640x480 and eight colors.
